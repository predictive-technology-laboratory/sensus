    EncryptSymmetrically(Byte[], Int32, Int32, String)

    Encrypts bytes asymmetrically via symmetric encryption. Since asymmetric encryption does not support large data sizes, the approach is to generate a symmetric encryption key that is designed for large data sizes, encrypt the data with the symmetric key, encrypt the symmetric key with the asymmetric key, and send the encrypted symmetric key and encrypted data to the same file.
