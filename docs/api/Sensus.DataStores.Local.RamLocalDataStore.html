Class RamLocalDataStore

When using the RAM Local Data Store, all data coming off the Probes are stored in RAM. Since RAM is volatile storage, any stored data will be lost if Sensus is turned off or crashes. Thus, most practical Sensus deployments will probably want to avoid this local data store, opting instead for the FileLocalDataStore.
