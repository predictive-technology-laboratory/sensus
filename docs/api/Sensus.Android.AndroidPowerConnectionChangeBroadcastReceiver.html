    Class AndroidPowerConnectionChangeBroadcastReceiver

    https://developer.android.com/reference/android/content/Intent#ACTION_POWER_CONNECTED that wish to register specifically to this notification. Unlike ACTION_BATTERY_CHANGED, applications will be woken for this and so do not have to stay active to receive this notification. This action can be used to implement actions that wait until power is available to trigger. added in API level 4

    We use the same receiver for both the connected and disconnected intents.
