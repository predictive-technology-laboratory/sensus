    TelephonyDatum

    Represents phone call meta-data in terms of the phones TelephonyState, the other phone number, and -- if a call just ended -- the duration of the call. Both IncomingCall and OutgoingCall values will be associated with an unspecified CallDurationSeconds, as the former are recorded to mark the time at which the incoming call arrived or went out, respectively. When a call ends and the phone returns to Idle, then there will be a value for CallDurationSeconds indicating how long the call lasted.
